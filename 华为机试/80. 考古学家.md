考古学家发现了一个石碑，可惜其已经断成了多段，原地发现n个断口整齐的石碑碎片，为了破解石碑的内容，考古学家希望有程序可以帮忙计算复原后的石碑文字的组合数。

输入描述：
第一行输入n，表示石碑碎片的个数
第二行依次输入石碑碎片上的文字内容s，共有n组

输出描述：
输出石碑文字的组合(按升序排列)，行末无多余空格

示例1：
输入
3
a b c
输出
abc
acb
bac
bca
cab
cba
说明
abc三个字母的所有组合，按照字典序升序排列输出

示例2：
输入
3
a b a
输出
aab
aba
baa
说明
可能组合有“aab”“aba”“baa”

示例3：
输入
3
a b ab
输出
aabb
abab
abba
baab
baba

备注：
如果存在石碑碎片完全相同，相同碎片间的顺序变换不影响复原后的碑文内容。
