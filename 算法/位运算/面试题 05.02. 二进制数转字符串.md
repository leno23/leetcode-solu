二进制数转字符串。给定一个介于0和1之间的实数（如0.72），类型为double，打印它的二进制表达式。如果该数字无法精确地用32位以内的二进制表示，则打印“ERROR”。

**示例1:**

```
 输入：0.625
输出："0.101"
```

**示例2:**

```
 输入：0.1
输出："ERROR"
提示：0.1无法被二进制准确表示
```

```py
# 小数的二进制表示为1/2^1 + 1/2^2 + 1/2^3,所以每次乘以2可以得到各项前面的系数，也就是最终的答案
class Solution:
    def printBin(self, num: float) -> str:
        out='0.'
        for i in range(6):
            num *=2
            if num<1:
                out+='0'
            else:
                num-=1
                out+='1'
                if num==0: return out
        # 6位之内不能完整表示
        return 'ERROR'
