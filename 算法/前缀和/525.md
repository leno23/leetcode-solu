525. 连续数组

给定一个二进制数组 nums , 找到含有相同数量的 0 和 1 的最长连续子数组，并返回该子数组的长度。

 

示例 1:

输入: nums = [0,1]
输出: 2
说明: [0, 1] 是具有相同数量 0 和 1 的最长连续子数组。

```js
var findMaxLength = function (nums) {
    let sum = 0, n = nums.length, m = new Map([[0, -1]]), res = 0
    for (let i = 0; i < n; i++) {
        sum += nums[i] ? 1 : -1
        if (m.has(sum)) res = Math.max(res, i - m.get(sum))
        else m.set(sum, i)
    }
    return res
};
