829. 连续整数求和

给定一个正整数 n，返回 连续正整数满足所有数字之和为 n 的组数 。

示例 1:

输入: n = 5
输出: 2
解释: 5 = 2 + 3，共有两组连续整数([5],[2,3])求和后为 5。

```js
/**
 * @param {number} n
 * @return {number}
 */
var consecutiveNumbersSum = function (n) {
    let res = 0,
        length = 1;
    // 枚举连续序列的每一种可能的长度
    while (n > 0) {
        // 1.连续序列长度=1 时,只有一个序列满足  就是n
        // 那么如果,序列的和为n，也就是 a+a+1=n --> a=(n-1)/2，因为a是正整数，所以需要满足a % (n-1) ==0
        // 3.连续序列长度=3 时,假设 a、a+1、a+2 满足条件
        // 那么 a+a+1+a+2=n --> a=(n-1-2)/2 --> a % (n-1-2) ==0
        // 4.连续序列长度=4 时,假设 a、a+1、a+2、a+3 满足条件,
        // 那么 a+a+1+a+2+a+3=n --> a=(n-1-2-3)/2 --> a % (n-1-2-3) ==0
        // 5....随着序列长度增加，第一个数在变小
        // 6.直到 第一个数 < 0，也就是 (n-1-2-...) < 0 时，即 n<0时， a为负数，不满足题目要求的正整数的要求

        if (n % length == 0) res++;
        // 此时，迭代中的n代表分子，由上面的推导可以知道，每一轮需要减去length得到下一轮新的分子
        n -= length;
        length++;
    }
    return res;
};
```
