258. 各位相加

给定一个非负整数 num，反复将各个位上的数字相加，直到结果为一位数。返回这个结果。

示例 1:

输入: num = 38
输出: 2
解释: 各位相加的过程为：
38 --> 3 + 8 --> 11
11 --> 1 + 1 --> 2
由于 2 是一位数，所以返回 2。

```js
/**
 * @param {number} num
 * @return {number}
 */
var addDigits = function (num) {
    // 1.一个非负整数可以表示为num = 100*a+10*b+c = 99*a+9*b+(a+b+c)
    // 各位相加也就是求a+b+c的值
    // 如果a+b+c不能被9整除，对于等式两边对9取余
    // 可以得到，num%9 = (a+b+c)%9.
    // 此时(a+b+c)这部分又可以看做是一个数字,同样重复1的过程
    // 那么最终的答案就是 num%9
    // 当然，有两种边界需要考虑
    if (num == 0) return 0;
    if (num % 9 == 0) return 9;
    return num % 9;

    // 也可以简写为
    // return (num-1)%9+1
};
```
