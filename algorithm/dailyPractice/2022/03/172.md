172. 阶乘后的零

给定一个整数 n ，返回 n! 结果中尾随零的数量。

提示 n! = n _ (n - 1) _ (n - 2) _ ... _ 3 _ 2 _ 1

示例 1：

输入：n = 3
输出：0
解释：3! = 6 ，不含尾随 0

```js
var trailingZeroes = function (n) {
    // n!阶乘中的0只会由2*5获得
    // 又因为n!中质因子2的个数要多于5，所以
    // 只需要求出其中因子5的个数即可，
    // 也就是求1,2,3,...,n中每个数中因子5的个数之和
    // 即求[5,n]中每个数中5的倍数的和
    let ans = 0;
    for (let i = 5; i <= n; i += 5) {
        for (let x = i; x % 5 == 0; x /= 5) {
            ++ans;
        }
    }
    return ans;
};
```
